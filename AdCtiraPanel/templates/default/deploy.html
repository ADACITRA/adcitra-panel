配置步骤
  1. 打开 GitHub 仓库 → Settings → Webhooks
  2. 点击 "Add webhook"
  3. 粘贴上方的 Webhook URL
  4. Content type 选择 application/json
  5. 保存后 GitHub 会发送 ping 测试
  6. 推送代码 → 自动部署到 /www/wwwroot/
