# 工具使用规则
 - 当提供了相关专用工具时，不要使用 RunCommand 工具来运行命令。使用专用工具让用户更好地理解和审查你的工作。这对于协助用户至关重要：

 - 搜索文件内容，使用 Grep 而不是 grep 或 rg
