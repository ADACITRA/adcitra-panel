git clone https://github.com/adcitra/panel.git

cd panel && pip install -r requirements.txt
python runserver.py
# 访问 http://localhost:8888

常见问题

和宝塔面板是什么关系？▼
AdCtira灯塔 基于宝塔Linux面板（aaPanel/BaoTa）开源代码复刻，进行了全品牌重命名、外链清理和功能升级。保留了宝塔强大的生态兼容性，300+ 应用可直接使用。
专业版和企业版什么时候上线？▼
核心功能现在完全免费使用。专业版和企业版正在开发中，预计 Q3 2026 开放。订阅用户将获得 AI 运维、云备份、企业安全策略等增值功能。
会不会像宝塔一样后来开始收费？▼
基础版将永远免费。这是我们和宝塔最大的不同——基础功能不阉割、不限制、不恶心用户。收费的是增值服务和云功能，而且价格会比宝塔合理得多。
有微信群或社区吗？▼
社区正在筹备中。目前可以通过 GitHub Issues 提交问题和建议。后续会建立论坛、微信群、Discord 等交流渠道。
