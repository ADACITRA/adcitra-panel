一个平台，六项核心能力

adcitra deploy

一行命令部署任何应用。自动检测框架、生成nginx配置、创建数据库。支持 Laravel/Django/Next.js 等9种框架。

Git 自动部署

连接 GitHub webhook，推送即部署。非 main 分支自动创建预览环境，提前验证变更。

服务器管理

网站/数据库/Docker/SSL/防火墙/文件 — 全套运维能力。300+ 应用一键安装，Nginx/MySQL/PHP 深度管理。

AI 自然语言运维

说人话就能管理服务器。网站访问慢帮我看看 — AI 自动诊断 CPU/内存/磁盘/网络。

一键迁移

从其他面板无缝迁移到灯塔。网站/数据库/FTP/SSL 全部自动转换，路径零手动操作。

极简模式

面板内存占用 < 50MB。适合 512MB 低配 VPS、树莓派、ARM 设备，友商越做越臃肿，我们反着来。

一分钟上手

Docker 部署，一行命令

# Docker 一键部署
$ git clone https://github.com/adcitra/panel.git
$ cd panel && docker compose up -d
OK 面板: http://localhost:8888
$ adcitra deploy
OK 应用已上线
